## 🎯 Escopo Técnica
O projeto foca no desenvolvimento de lógica utilizando PHP, com ênfase em:
* **Estruturas Condicionais:** IF/ELSE e Switch (Ex: Intervalo, Ano Bissexto).
* **Laços de Repetição:** For/While (Ex: Fibonacci, Triângulo Numérico).
* **Funções e Vetores:** Modularização de código (Ex: Soma de 0 a N, Média Aritmética).

---

## 📂 Organização dos Exercícios
Seguindo as normas da atividade, cada exercício possui sua própria pasta:

* **ex01/** - Intervalo entre 100 e 200
* **ex02/** - Conversão de Temperatura (C/F)
* **ex03/** - Calculadora Aritmética
* **ex04/** - Triângulo Numérico (n linhas)
* **ex05/** - Somatório de Fatoriais
* **ex06/** - Série de Fibonacci
* **ex07/** - Vetores: Positivos e Negativos
* **ex08/** - Função: Soma de 0 até N
* **ex09/** - Função: Média Aritmética
* **ex10/** - Cálculo de Ano Bissexto
